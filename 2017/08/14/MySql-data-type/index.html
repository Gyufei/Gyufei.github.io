MySql 的数据类型

发表于 2017-08-14 | 分类于 语言 ， SQL |
字数统计 340 字 | 阅读时长 1 分钟

MySql 作为常见的数据库，有多种数据类型，用于存储不同类型的一些数据。

本文主要对 MySql 的数据类型进行了一些介绍和总结。

数值类型

整数

  • TINYINT 小整数值 （0-255 or -128-127) 1字节

  • SMALLINT 大整数值 2字节

  • MEDIUMINT 大整数值 3字节

  • INT 或 INTEGER 大整数值 4字节

  • BIGINT 极大整数值 8字节

浮点数

  • FLOAT 单精度浮点数值 4字节
  • DOUBLE 双精度浮点数值 8字节
  • DECIMAL 小数值

字符串类型

字符字符串（存储过程中不进行大小写转换)

  • CHAR 定长字符串 0-255字节
  • VARCHAR 变长字符串

字节字符串（二进制字符串）

  • TINYBLOB 不超过255字节的二进制字符串
  • BLOB 长二进制字符串
  • MEDIUMBLOB 中等长度二进制字符串
  • LONGBLOB 极大长度二进制字符串

文本

  • TINTTEXT 短文本字符串
  • TEXT 长文本字符串
  • MEDIUMTEXT 中等长度文本字符串
  • LONGTEXT 极大长度文本字符串

日期和时间类型

年

YEAR 年份值（ YYYY ） 1字节

日期

DATE 日期值（YYYY-MM-DD） 3字节

时间

TIME 时间值 （HH:MM:SS) 3字节

日期和时间

DATETIME 混合日期时间值 （YYYY-MM-DD HH:MM:SS）8字节

时间戳

TIMESTAMP 时间戳字符串 （YYYYMMDD HHMMSS）4字节
