Momnet.js API 文档翻译整理

发表于 2018-09-08 | 分类于 前端 |
字数统计 4,973 字 | 阅读时长 21 分钟

Moment.js是一个用来处理日期和时间的JS库，功能相当强大，为日常开发中的时间相关处理提供了很多便利。

Moment.js模仿了jQuery的链式调用方法和合并getter和setter方法，使用起来还是十分的高效直观的。各种形式提供便利的API也不少。

但官网上的文档相当省事，只是把标题翻译成了中文就当做中文文档了。你说你要全英文我也就不说了，你光把标题和章节名翻译成中文然后就说这是中文文档然后放在官方中文网也太敷衍了把！！要逼死强迫症吗？

所幸这个库的API还算比较简明和直观，并没有太多的需要费力理解的地方，需要翻译的地方也不算特别多，因此我就花了一天时间，把它的文档翻译整理了一下，便于以后查阅，也顺带看一下在以前的使用过程有没有哪些功能没有涉及到或者注意到。

偷懒了一下，关于i18n国际化的自定义设置语言这一部分，只是简单的总结了一下，并没有详细的翻译，因为并没有多少机会能用到。。。。

Moment.js是一个用于处理Js的日期和时间的类库。适用于IE8以上浏览器环境和node环境。

构造函数

Moment.js将Date对象包装成一个moment对象从而实现各种操作和处理。
构造函数moment()，用来创建一个moment对象。类似于new Date()方法，但可以接收更多的参数格式。

静态方法

moment.utc()

Moment默认使用本地时间。
使用moment.utc()创建一个UTC模式的moment对象，这个对象的所有时间操作和展示都会使用UTC模式。
方法的参数与moment()构造函数相同。

moment.unix(Number)

从一个Unix的秒级时间戳中创建一个moment对象

moment.parseZone(string)

进行时区设置。

moment.max(Moment[,Moment])

返回接收的所有moment实例参数中的最大值

moment.min(Moment[,Moment])

返回接收的所有moment实例参数中的最小值

moment.isMoment(obj)

moment.isMoment(obj)用于检查传入的参数是否是moment对象。

moment.isDate(obj)

moment.isDate(obj)用于检查传入的参数是否是Js的Date对象。

moment.duration

用于创建时间段，关于时间段的具体操作可以查看实例方法中的时间段一节。
有以下四种调用方式

moment.isDuration(obj)

moment.isDuration(obj)用于检验对象是否为有效的时间段对象

moment.normalizeUnits(string)

获取传入字符串的时间单位全名

实例方法

moment.fn代理了moment的原型，可以通过其来为moment添加自定义的实例方法。同时moment对象使用了连接接口模式，因此我们可以对它的实例方法进行链式调用。

检验

moment().isValid()
检验moment对象的有效性，返回布尔值。

操作

加操作

moment().add()用于时间增加操作。有以下三种调用方式:

  • moment().add(Number, String)

可以使用以下这些时间单位进行对应加减, 他们均有对应的简写模式:

  • moment().add(Duration)

  • moment().add(Object)

当增加月时，如果目标的号天数超出当月最大值，则回变为目标月最后一天

减操作

moment.subtract()跟moment.add()具有相同的调用方式和调用参数，不同的只是它用来减少时间。

开始时间

moment.startOf()将moment的时间设置为某个时间单位的起始时间。

结束时间

moment.endOf()将moment的时间设置为某一时间单位的结束时间。与moment.startOf相似。

最大值

moment().max(Moment)将当前moment与另一个moment比较，并取两者中最大的值作为当前moment的值。

最小值

将当前moment与另一个moment比较，并取两者中最小的值作为当前moment的值。

本地化

moment().local()用于标示moment内部使用本地时间模式

UTC

moment().UTC()用于表示moment内部使用UTC模式。

当月天数

moment().daysInMonth()返回当前moment对象所在月的天数

时间差

moment.fromNow()
moment().fromNow()用于显示moment对象的对应时间距现在已经过去了过久。

如果向fromNow传递一个布尔值true，则格式化出的时间差不带后缀词。

moment.toNow()
moment().toNow()用于显示moment对象的对应时间距现在还要多久。
与fromNow类似，它也接受一个布尔值true为参数，从而不带后缀词。
显而易见，m.fromNow() = - m.toNow()。

moment.from(moment-like)
接受一个时间格式参数，并返回其距当前moment对象时间已过去了多久。
同样可以接受第二个true参数来取消后缀。
moment-like参数可以是一个moment对象，也可以是时间字符串或者 js 的 Date对象或者时间数组。

moment.to(moment-like)
与from方法其他的都相同，只是返回接受的时间格式参数距现在还要多久。

moment.diff(moment-like)
接受一个时间格式参数，返回其和当前moment对象的毫秒为单位的事件差。
可以接受第二个参数，用于定义返回的时间差的单位。
可以接受第三个参数为true，则取消返回值的后缀词。

转换值

JS Date 对象

moment().toDate()可以得到当前moment包装的Js Data对象。
需要注意的是浅复制，因此对Date对象的改变都会反映在moment对象上。
如果想要避免这种情况，可以在toDate方法之前先调用momnet().clone()方法。

Array

moment().toArray()返回一个数组，即new Date()参数的镜像。

####对象
moment().toObject()返回一个包括年月日时分秒毫秒的对象。

JSON

moment.toJSON()
将moment对象序列化为JSON。

ISO 8601字符串

moment().toISOString()返回一个ISO 8601的字符串

全局语言设置

使用静态方法moment.locale()来进行全局的语言设置。

局部语言设置

使用实例方法`moment().locale(string)来进行单个实例的语言设置。

在momentJs 2.8.1版本之前的语言设置，需要通过moment.lang()和moment().lang()。
在浏览器中加载语言只需要引入对应的类似locale/zh.js。语言文件即可。
在Node.js中则需要保证在moment-root/locale中存在需要加载的语言文件，然后使用moment.locale()来加载。
