VimScript入门(一):编辑器相关

发表于 2018-07-06 | 分类于 VIM ， 编辑器 |
字数统计 1,405 字 | 阅读时长 5 分钟

用 VIM 也有一段时间了，插件和各种配置也尝试了不少，但由于不懂VimScript，很多时候遇到问题，并没有办法自己解决，只能去各种搜解决方案。很多时候，解决了问题，也是知其然而不知其所以然。

因此，打算学习一下VimScript，从而加深对Vim的了解，也可以在后续根据实际需求来自己定义一些Vim配置，写一些实用的Vim脚本。

这篇是 VimScript入门系列 的第一篇。

输出

"打印消息到命令行
ec
echo

另外还有echo的各种形式，例如echom 以消息形式打印，echoe以错误形式打印。

注释

" 使用一个双引号作为注释

选项设置

布尔选项

以 name 选项为例

" 将选项设置为on
set <name>

" 将选项设置为off
set no<name>

" 切换选项值
set <name>!

" 查询选项值
set <name>?

键值选项

对于选项值不是布尔值而是其他具体值的选项，使用set，如果想只作用于当前文件，使用setlocal。

" 设置选项值
set <name>=<value>

" 查询选项值
set <name>?

" 可以一次设置多个选项值
set <name1> <name2>=<value>

一些基本的设置选项

" 布尔选项
number " 是否显示行号，
wrap   " 是否折行
shiftround " 使用 >> 或 << 时， 是否依shiftwidth 调整宽度
relativenumber " 是否使用相对行号

numberwidth "行号的宽度，键值选项
matchtime "括号高亮的匹配时间

映射

" 将某按键name映射为<key>按键
map <name> <key>

" 删除按键
unmap <name>

"非递归的映射
noremap <name> <key>
  • 以上三种映射均可以在前面加上i，n,v来表示插入，普通和可视模式。
  • Vim可以映射按键序列，形如noremapkey
  • 有以下几个特殊按键
    • <C-d> Ctrl
    • <A-d> Alt
    • <space> 空格键
    • <esc> Esc键
    • <nop> 无键值，通常用于取消掉一些默认按键的行为
  • leader 键 可以通过let mapleader=设置键name为leader，来更好的进行按键映射。
    • 另外有一个localleader选项，可以用于特定文件中的leader键设置。
  • 键盘映射无法使用 “ 在行内注释。
  • 可以使用buffer设置只作用于当前文件的本地映射，形如:nnoremapQ x 。
  • :normal 用于在定义命令时相当于在普通模式下按下其后跟随的按键。但它无法识别特殊字符。
  • 如果想识别特殊字符，比如尖括号，下划线等，可以使用excute命令。excute命令后跟随一个 vim 的字符串形式脚本命令，类似这样excute "normal! gg"

operator-Pending 映射

在VIM中，可以在一个操作<operator>后跟随一个移动命令或选中命令，从而为这个操作指定范围。比较常见的操作例如d，y,c等。
可以使用operator-Pending映射来将一个移动命令定义为operator-pending映射。从而使其可以与操作命令连用。
如这个命令：
onoremap p i(
就是将 p 映射到 i(, 使用dp命令，就相当于执行了di(

通常，operator-Pending映射的作用规则是:

  • 如果映射到的命令以选中文本结束，则操作命令会操作这些文本。
  • 否则，VIM 会从光标位置操作到移动命令终点之间的文本。

缩写abbreviations

使用abbrev选项来定义缩写，在你输入某些字符时自动替换为另外的字符。一般在前面加上i来在插入模式下使用，如下：

" 当你输入 retrun 时自动替换为 return
iabbrv retrun return

缩写可以用来纠正常见的拼写错误或者定义一些简短的代码片段。

自动命令

自动命令，可以在特定模式的特定事件事件发生时，自动执行命令。

使用autocmd来编写自动命令。

形如：

autocmd VimEnter * PlugInstall

具体如下：

  • vimEnter为事件类型，可以使用逗号来添加多个事件到自动命令。使用:help autocmd-events来查看 VIM 支持的所有事件类型。
  • 为事件匹配模式pattern，可以是文件名，文件类型等
  • PlugInstall为自动执行的命令

FileType 事件

FileType 事件会在 VIM 设置一个缓冲区的时候filetype时触发，是非常常用的事件。

下面是一个在文件类型为 Python 时自动将<leader> c设置为添加行注释的命令

:autocmd FileType python nnoremap <buffer> <localleader>c I#<esc>

自动命令组

因为 VIM 在每次读入.vimrc文件时，都会载入自动命令，导致命令被多次定义。可以使用命令组的自动清除来防止这个问题。

命令组的定义如下：

:augruop groupname
:autocmd ...
:autocmd ...
:augruop end

通过在命令组中加入清除autocmd命令autocmd !，来实现每次载入命令组时都清除组内的所有自动命令效果。

:augruop groupname
:autocmd ！
:autocmd ...
:augruop END

状态条

使用set statusline=来设置状态条的格式，一些有用的字符串插值如下:

  • %f 文件名
  • %F 文件路径
  • %y 文件FileType类型
  • %l 当前行号
  • %L 总行数
  • %= 切换到状态条右侧

有以下几个需要注意的地方：

  1. 空格需要使用\转义
  2. 使用数字来表示字符最小宽度
    • %4l，表示行号最少占4个字符。
    • 空位字符位于右边，则使用%-4l。
    • 使用0代替空位，则使用%04l。
  3. 使用点加数字来表示字符最大宽度。形如%.20F

可以使用+=来通过多行代码定义动作条，使其语义更清晰。
